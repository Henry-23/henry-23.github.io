VSCode快捷键笔记

发表于 2020-07-10 更新于 2022-07-30 分类于 笔记
大一暑假在市图书馆学VSCODE快捷键，还记得当时为准备阿萨姆班的面试挺紧张的。

Visual Studio Code 快捷键学习笔记

1.向左缩进(indent)

使用crtl+[来实现向左缩进,另一个括号能实现向右,但没啥用但可以用Tab代替

2.选中多行

使用shift+方向键实现多行选中

3.上下移动

使用ALT+方向键实现上下移动(可配合2中的快捷键实现多行移动)

4.整行复制

使用shift+ALT+方向键(上下)实现整行复制

5.编辑点回退

xxxxxxxxxx $ google-chrome –app=https://www.notion.sobash

6.对{}的操作

1.收起或展开花括号内的代码

使用Ctrl+shift+[或]实现

2.跳转到另一个花括号

使用Ctrl+shift+\实现

3.花括号内容的折叠(这有助于快速了解代码格局)

  • 发音相似:

collapse(折叠)->K

7.查找/替换

使用Ctrl+F查找
使用Ctrl+H替换

8.打开调试面板

使用Ctrl+shift+D(debug)

9.唤起终端

使用Ctrl+`打开终端
使用Ctrl+shift+` 新建终端
使用Ctrl+shift+5拆分终端

10.命令窗口

输入setting,launch等能打开相应的json文件

11.文本操作快捷键

Shift + ← / → (方向键)： 光标向左/向右选中一个字符
Ctrl + Shift + ← / → (方向键)： 光标向左/向右选中一个单词
Shift + Home： 光标从当前位置一直选中到行首
Shift + End： 光标从当前位置一直选中到行尾

Home： 光标跳转至行首
End： 光标跳转至行尾
PgUp： 光标跳转至首行
PgDn： 光标跳转至末行
Ctrl + ← / → (方向键)： 光标向左/向右跳转一个单词
Ctrl + Home： 光标跳转至首行首个字符前面(文件开头)
Ctrl + End： 光标跳转至末行最后一个字符后面(文件结尾)
