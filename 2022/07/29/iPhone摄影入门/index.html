1.2 注意力引向主题

三分法是在一张照片中，我们两横两纵，然后把画面中的重点放在画面的焦点上。这样做的本质原因是，我们人的视线在看一张画面的时候会优先看到这些焦点区域。

2. 硬件

13pro有三颗镜头，点选快门键旁边的焦距倍数能获得最好的分辨率，适时采用2倍，1.5倍这些约定于35mm和50mm的焦段也能拍出很好的效果

2.1 超广角镜头

由于超广角镜头的天然畸变，所以在靠近画面边缘的部分造成拉长的效果。相反，在画面中央的部分，就会有着压缩的效果。因此我们如何控制好画面中的人物站位，就决定了人物的身材比例。

在建筑和风光摄影中，也会看到畸变带来的效果。当我们想要表像一座建筑的雄伟的时候，就可以利用畸变拉长。当我们想要表现一座建筑或者是山峰的渺小的时候，就可以放置在画面的中央。

2.2.1 用来拍人

  • 把人脸放在绝对中央
  • 仰拍，让被拍者向下看，望向镜头

2.2.2 用来拍景

  • 仰角度：显得建筑宏大

  • 竖构图：线条延展修长

  • 放大前景：超广角的镜头特点：能突出前景

2.2 长焦镜头

2.2.1 使用16：9比例

与5倍长焦相比，3倍长焦在拍摄建筑时画面就没有那么简洁紧凑，经常会把无关的元素带入画面之中。

这时候我建议你切换至16:9的比例来拍摄，16:9相比4:3会有一定的“放大”作用，让画面看起来更加简洁。

2.2.2 骗过系统

很多时候，由于场景限制，iPhone会选择裁剪普通镜头的方式来实现变焦，并不调用长焦镜头，这一点可以在图片的分辨率中发现。

解决：把手机对准身边的光线比较强的物体，这时候就切换到长焦了，再对准拍摄物体

3. 软件

3.1 自动曝光

iPhone会自动调整正确的曝光，但是正确的曝光不一定是一张好好看的照片，比如下面的一些照片都是光线不够充足的「低调照片」。但你会发现他们有着独特的美感：

大多数手机摄影爱好者对于曝光还停留在拍摄的足够亮，似乎画面中所有细节都在就是好照片。

但是照片其实和音乐一样，是有所谓的「影调」的。亮的照片只是其中一种高调作品。但是在摄影家眼里，一般都是低调作品更加的耐看，或者是高级。

我们在曝光的时候，最简单的提高质感的方式，就是还原照片中的光线。通过控制曝光，让照片中的层次丰富。

尝试一些低曝光的低调作品，反而可能会更加的好看。

3.2 滤镜

银色调：胶片质感

反差色：冷淡风

3.3 摄影风格

在iPhone 13系列中，相机菜单新增了一项「摄影风格」的功能，与普通滤镜最大的不同是它能与硬件更好的融合。四种风格分别是：高饱和度、高对比度、暖色调、冷色调。

3.3.1 高饱和度

照片色彩会明亮生动，适合拍摄色彩表现力强的题材，例如花卉、车辆、红墙、蓝天等突出色彩的题材；

3.3.2 高对比度

主要特点是画面明暗反差大，在杂乱的场景中“高对比度风格”能让画面变得更加简洁。

3.3.3 暖色调

暖色调对画面中橙色部分比较敏感，在日出日落之时适合采用暖色调来强调氛围。

3.3.4 冷色调

冷色调却恰恰相反，像阴雨天、雪地、傍晚的蓝调时刻，用冷色调拍摄能更好的烘托出建筑的冷峻。

3.4 人像模式

想要最佳的效果，一般都是在 ƒ/2.8 左右，这样的画面一般都是真实的。不过拍摄时不需要选择最佳效果，都可以后期编辑照片进行调整。

注意：不适合拍摄全身照

3.4.1 虚化技巧：

1. 人物和背景一定要保持一定的距离，最好是有一定的空间纵深感。

2. 学会看取景框。

是不是足够拍摄的条件，取景框中会显示。当出现黄色对焦框的时候，就说明已经可以进行人像虚化拍摄。

3. 人像模式在其他静物中使用，会有意想不到的效果。

4. 想要好看的光斑焦外，要保证焦外本身有点状光源。

3.4.2 光效选择

  • 摄影室灯光：最为简单的说法就是给你的脸补光。这种补光可以让你的肤色变得非常的透亮

  • 轮廓光：反差比较强烈，硬朗，在光线硬朗环境下，轮廓光会直接强化你这种质感。

人物至少要在腰以上的画面，如果是全身人像，效果不佳。

3.5 夜景模式

核心：最佳拍摄时间不是在乌漆嘛黑的夜晚，而是日落前后

夜景模式能够在光线不足的情况下尽可能还原肉眼的色彩。

在不是夜景的时候使用，减少噪点，提高锐化对比

自然弱光、静止物体：夜景时间拉满，画质能最高

3.6 画幅

3.6.1 4:3比例：人像、风光

默认照片比例是4:3，因为这个比例更接近传感器的比例。4:3的画幅比例可以拍摄视野更广、动态范围更广的照片，后期可塑性更高，适合人像、风光拍摄；

3.6.2 16:9比例：大的风光场景

＂长条型＂的横屏16：9适合展现大的风光场景，竖屏16：9侧拍摄高大的对象，例如高大的建筑。

3.6.3 1:1比例：食物、植物

适合中心点构图，适合拍摄食物、植物等题材；

4.拍人

4.1 构图

  • 半身像：胸部切分

  • 一般头顶留一到两拳，七分人像可以更多

  • 不要在关节处切割

  • 用好网格线，2/3

4.2 把握光线

  • 夜晚可以使用强制闪光灯补光

  • 学会观察环境灯光，冷暖色调、夜景模式、降低曝光补偿来不触发夜景模式以及减小hdr力度

4.3 打卡照拍出新意

核心：观察周围环境，动态选择拍照点（即路人不会拍照的地方）
